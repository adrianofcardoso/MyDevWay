# VSCode
Aqui armazeno os principais atalhos para produtividade.


## **Atalhos**

Atalho do teclado para mudar varios elementos de uma vez só:

- Shift+ ALt + I = muda o nome de todas as variáveis selecionadas
- Ctrl + Shift + -> = Seleciona todas as palavras da linha a direita
- Ctrl + D = Você seleciona o elemento e depois com Ctrl + D você seleciona o próximo elemento igual ao que você selecionou e assim por diante, até selecionar todos os elementos iguais
